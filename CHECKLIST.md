## 🐛 常见问题排查

### 如果 Gradle 同步失败
1. 检查网络连接
2. 尝试使用国内镜像（见 QUICKSTART.md）
3. 清除缓存：File -> Invalidate Caches / Restart
4. 删除 .gradle 文件夹后重试

### 如果编译错误
1. 查看 Build 窗口的具体错误信息
2. 检查是否所有依赖都已下载
3. Clean Project -> Rebuild Project
4. Sync Project with Gradle Files

### 如果运行时崩溃
1. 查看 Logcat 的错误堆栈
2. 检查是否有 NullPointerException
3. 确认 GameState 已正确初始化
4. 检查是否有资源文件缺失

### 如果 UI 显示异常
1. 检查 XML 布局文件是否有错误
2. 确认所有 view ID 与 Java 代码匹配
3. 检查 RecyclerView 的 Adapter 是否正确设置
4. 确认数据列表不为 null
